# AirPlay-Pi
AirPlay Orange Pi Zero
----------------------
-----------------------------------------------------------------------------------------------------------------------------

Commands :
//////////

* Run on Boot :
sudo systemctl enable shairport-sync

* Start AirPlay Service: 
sudo service shairport-sync start


* Check AirPlay Service Status: 
sudo systemctl status shairport-sync.service
